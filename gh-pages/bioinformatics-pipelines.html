5 Bioinformatics Pipelines

5.1 Pipeline for analyzing 16S rRNA amplicon data

Below are few most popular platforms that offers free lightweight and reproducible tutorials for analyzing 16S rRNA gene sequencing data.

Pipeline Description Tutorial
Mothur An open-source software package for bioinformatics data processing including 16S rRNA gene sequencing data Basic Tutorial
QIIME2 An open-source microbiome analysis package including 16S rRNA gene sequencing data Tutorials
Nephele Open, accessible high performmance cloud platform platform for microbial bioinformatics. Integrates Mothur, QIIME2, DADA2 and Biobakery Some tutorials

Make sure to check for latest instructions online.
